# Pi Sky Tracker v1.0 🚀

Aplikasi hybrid: Game + Tracker + Marketplace untuk Komunitas Pi Network

## Fitur
- **My Tracker**: Lihat saldo Pi game + statistik + skor tertinggi
- **Pi Shooter Game**: Game pesawat tembak. Kendali swipe. 7 nyawa, naik level
- **Marketplace**: Beli Extra Nyawa, Skin Emas, Perisai pake Pi Game
- **Swipe Navigation**: Geser kiri/kanan untuk pindah halaman
- **Responsive**: Bisa main di HP & PC lewat Chrome

## Cara Main - Versi Chrome
1. Buka `index.html` di browser
2. Swipe Kanan = Masuk Game | Swipe Kiri = Masuk Market
3. Kumpulkan Pi dengan menembak musuh
4. Pakai Pi untuk beli item di Market
